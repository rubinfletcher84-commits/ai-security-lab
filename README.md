# AI Security Lab

This project documents my first hands-on AI security lab built in an Ubuntu virtual machine using VirtualBox, Ollama, and Garak.

## Project Overview

I created a local AI security testing environment to better understand how large language models can be evaluated for vulnerabilities and unsafe behavior.

## Tools Used

- VirtualBox
- Ubuntu
- Python virtual environment
- Ollama
- Garak

## What I Did

- Built a fresh Ubuntu VM in VirtualBox
- Created a Python virtual environment
- Installed Garak, an LLM vulnerability scanner
- Installed Ollama and ran local models
- Connected Garak to a local model
- Ran my first AI security scan
- Generated HTML and JSON scan reports
